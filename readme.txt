== Description ==

Add to orkut is a simple plugin developed using orkut share api.

This plugin will give your users an option to share content on your website on thier orkut profiles

The shared content will be shown as updates and users can also promote it on orkut to thier friends (visit http://way2wp.com for screenshots)

Hope you like this plugin .. It helps you get Traffic as well as Promotion

Enjoy !!!

== Installation ==

1.  Add a directory called add-to-orkut to /wp-content/plugins/ and unzip the files there.
2. Activate the plugin through the Plugins menu in WordPress, click settings to change options.


visit http://way2wp.com/add-to-orkut-wordpress-plugin-104 for detailer install instructions
